# 🎮 Valorant Analytics Dashboard

An interactive data analytics dashboard built with Streamlit to analyze real Valorant game data.

## 🚀 Features

- 📊 Player performance analysis
- 🔫 Kill & KDA metrics
- 🎮 Agent-based insights
- 🏆 Rank distribution
- 🆚 Player comparison
- 🎛️ Interactive filters

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- SQLite

## 📂 Dataset

This project uses real-world game data stored in a SQLite database.

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
